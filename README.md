# Export-Sticky-Notes-to-.md-or-.txt-Files
**Options to include markdown's and last edited date.**  
**Comes with a mini Command Prompt UI**

## Requirements:

**PowerShell 5.1 or above** *(Run as Administrator)*

**[SQLite Tools](https://www.sqlite.org/download.html)**  
**Install the `sqlite-tools-win-x64-LATESTVERSION#.zip`**  
**Or just use this [Direct Download Link](https://www.sqlite.org/2025/sqlite-tools-win-x64-3500400.zip)**

## How to Use:

- **Create a folder in `C:\` named `CleanPaths`** 

- **Create a folder inside `CleanPaths` called `sqlite`** 

- **Unzip the SQLite download & place all the SQLite files inside `C:\CleanPaths\sqlite`**

- **Run the `launch.bat`**
